# Changelog yusa version v0.1.0

### v0.1.0 - resonance-logs-cn-yusa

#### 新功能
- **Game Overlay 重构**
  - 将独立的 attr-monitor 窗口整合到 game-overlay 窗口
  - 重构属性面板（AttrPanel）：实时显示角色战斗属性
  - 重构 Buff 组监控（BuffGroup）：支持分组显示和优先级设置
  - 新增怪物血量面板（MonsterHealthPanel）：显示怪物血量信息

- **重构 Overlay 设置页面**（`/main/overlay-settings`）
  - 基础设置（BasicSettings）：窗口透明度、缩放等
  - Buff 分组管理（BuffGroupManager）：支持分组配置
  - Buff 管理器（BuffManager）：支持搜索、优先级配置
  - Buff 监控设置（BuffMonitor、BuffSettings）
  - 技能监控设置（SkillMonitor、SkillSettings）
  - 配置文件管理（ProfileManager）：支持多配置切换
  - 快捷键设置（ShortcutSettings）
  - 怪物血量设置（MonsterHpSettings）
  - 全局优先级设置（GlobalPriority）
  - 单独监控全部 Buff（IndividualMonitorAll）

#### 更改
- **模块优化器**
  - CompactSolution 从 4 模组扩展为 5 模组支持
